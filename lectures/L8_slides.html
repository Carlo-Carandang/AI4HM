$k$-Means

  • $$\text{Minimize } \sum_{\ell=1}^k W(C_k)$$
  • $$\text{subject to } \bigcup_{\ell=1}^k C_k = [n] \text{ and } C_\ell \cap C_{\ell'} = \emptyset;$$
  • $$\text{for } W(C) = \frac{1}{|C|} \sum_{i,j\in C} \|x_i-x_j\|^2.$$
